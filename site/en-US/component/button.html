Button ​

Use Button for primary, secondary, and lightweight actions on mobile pages.

Basic usage ​

Use type, plain, round, dashed and circle to define Button's style.

vue
<template>
  <div class="button-example">
    <div class="button-row">
      <el-button>默认按钮</el-button>
      <el-button type="primary">主要操作</el-button>
      <el-button type="success">提交成功</el-button>
      <el-button type="info">补充信息</el-button>
      <el-button type="warning">提醒用户</el-button>
      <el-button type="danger">删除内容</el-button>
    </div>

    <div class="button-row">
      <el-button plain>朴素按钮</el-button>
      <el-button type="primary" plain>主要操作</el-button>
      <el-button type="success" plain>提交成功</el-button>
      <el-button type="info" plain>补充信息</el-button>
      <el-button type="warning" plain>提醒用户</el-button>
      <el-button type="danger" plain>删除内容</el-button>
    </div>

    <div class="button-row">
      <el-button round>圆角按钮</el-button>
      <el-button type="primary" round>主要操作</el-button>
      <el-button type="success" round>提交成功</el-button>
      <el-button type="info" round>补充信息</el-button>
      <el-button type="warning" round>提醒用户</el-button>
      <el-button type="danger" round>删除内容</el-button>
    </div>

    <div class="button-row">
      <el-button dashed>虚线按钮</el-button>
      <el-button type="primary" dashed>主要操作</el-button>
      <el-button type="success" dashed>提交成功</el-button>
      <el-button type="info" dashed>补充信息</el-button>
      <el-button type="warning" dashed>提醒用户</el-button>
      <el-button type="danger" dashed>删除内容</el-button>
    </div>

    <div class="button-row">
      <el-button :icon="Search" circle />
      <el-button type="primary" :icon="Edit" circle />
      <el-button type="success" :icon="Check" circle />
      <el-button type="info" :icon="Message" circle />
      <el-button type="warning" :icon="Star" circle />
      <el-button type="danger" :icon="Delete" circle />
    </div>
  </div>
</template>

<script lang="ts" setup>
import Check from '~icons/ep/check'
import Delete from '~icons/ep/delete'
import Edit from '~icons/ep/edit'
import Message from '~icons/ep/message'
import Search from '~icons/ep/search'
import Star from '~icons/ep/star'
</script>

<style scoped>
.button-example {
  display: flex;
  flex-direction: column;
  gap: 1rem;
}

.button-row {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
  align-items: center;
}

.button-row > * {
  margin: 0;
}
</style>

Disabled Button ​

The disabled attribute determines if the button is disabled.

Use disabled attribute to determine whether a button is disabled. It accepts a Boolean value.

vue
<template>

Link Button ​

WARNING

type="text" has been deprecated, and will be removed in 3.0.0, consider switching to new API.

New API link has been added in 2.2.1, you can use type API to set the theme of your link button

基础链接按钮

禁用链接按钮

vue
<template>
  <p>基础链接按钮</p>
  <div class="mb-4">
    <el-button
      v-for="button in buttons"
      :key="button.text"
      :type="button.type"
      link
    >
      {{ button.text }}
    </el-button>
  </div>

  <p>禁用链接按钮</p>
  <div>
    <el-button
      v-for="button in buttons"
      :key="button.text"
      :type="button.type"
      link
      disabled
    >
      {{ button.text }}
    </el-button>
  </div>
</template>

<script setup lang="ts">
const buttons = [
  { type: '', text: '默认' },
  { type: 'primary', text: '主要' },
  { type: 'success', text: '成功' },
  { type: 'info', text: '信息' },
  { type: 'warning', text: '提醒' },
  { type: 'danger', text: '删除' },
] as const
</script>

Icon Button ​

Use icons to add more meaning to Button. You can use icon alone to save some space, or use it with text.

Use the icon attribute to add icon. You can find the icon list in Element Plus icon component. Adding icons to the right side of the text is achievable with an <i> tag. Custom icons can be used as well.

vue
<template>
  <div class="button-row">
    <el-button type="primary" :icon="Edit" />
    <el-button type="primary" :icon="Share" />
    <el-button type="primary" :icon="Delete" />
    <el-button type="primary" :icon="Search">搜索</el-button>
    <el-button type="primary">
      上传资料<el-icon class="el-icon--right"><Upload /></el-icon>
    </el-button>
  </div>
</template>

<script setup lang="ts">
import Delete from '~icons/ep/delete'
import Edit from '~icons/ep/edit'
import Search from '~icons/ep/search'
import Share from '~icons/ep/share'
import Upload from '~icons/ep/upload'
</script>

<style scoped>
.button-row {
  display: flex;
  flex-wrap: wrap;
  gap: 0.75rem;
}
</style>

Button Group ​

ButtonGroup now stacks buttons vertically by default, which is more suitable for mobile action lists. Use direction="horizontal" only for compact bottom bars or paired decisions.

Use <el-button-group> to group related actions.

默认纵向，适合操作列表。
如果是底部双按钮栏，可显式设置为横向。
vue
<template>
  <div class="group-example">
    <el-text size="small">默认纵向，适合操作列表。</el-text>
    <el-button-group>
      <el-button type="primary" :icon="House">进入首页</el-button>
      <el-button type="primary" :icon="Operation">筛选条件</el-button>
      <el-button type="primary" :icon="Notification">消息提醒</el-button>
    </el-button-group>

    <el-text size="small">如果是底部双按钮栏，可显式设置为横向。</el-text>
    <el-button-group direction="horizontal">
      <el-button :icon="ArrowLeft">稍后处理</el-button>
      <el-button type="primary">
        立即提交<el-icon class="el-icon--right"><ArrowRight /></el-icon>
      </el-button>
    </el-button-group>
  </div>
</template>

<script setup lang="ts">
import ArrowLeft from '~icons/ep/arrow-left'
import ArrowRight from '~icons/ep/arrow-right'
import House from '~icons/ep/house'
import Notification from '~icons/ep/notification'
import Operation from '~icons/ep/operation'
</script>

<style scoped>
.group-example {
  display: flex;
  flex-direction: column;

Loading Button ​

Click the button to load data, then the button displays a loading state.

Set loading attribute to true to display loading state.

TIP

You can use the loading slot or loadingIcon to customize your loading component

ps: loading slot has higher priority than loadingIcon

vue
<template>
  <el-button type="primary" loading>加载中</el-button>
  <el-button type="primary" :loading-icon="Eleme" loading>提交中</el-button>
  <el-button type="primary" loading>
    <template #loading>
      <div class="custom-loading">
        <svg class="circular" viewBox="-10, -10, 50, 50">
          <path
            class="path"
            d="
            M 30 15
            L 28 17
            M 25.61 25.61
            A 15 15, 0, 0, 1, 15 30
            A 15 15, 0, 1, 1, 27.99 7.5
            L 15 15
          "
          />
        </svg>
      </div>
    </template>
    同步数据
  </el-button>
</template>

<script lang="ts" setup>
import Eleme from '~icons/ep/eleme'
</script>

<style scoped>
.el-button .custom-loading .circular {
  margin-right: 0.375rem;
  width: 1.125rem;
  height: 1.125rem;
  animation: loading-rotate 2s linear infinite;
}
.el-button .custom-loading .circular .path {
  animation: loading-dash 1.5s ease-in-out infinite;
  stroke-dasharray: 90, 150;
  stroke-dashoffset: 0;
  stroke-width: 0.125rem;
  stroke: var(--el-button-text-color);
  stroke-linecap: round;
  fill: transparent;
}
</style>

Sizes ​

Besides default size, Button component provides three additional sizes for you to choose among different scenarios.

Use attribute size to set additional sizes with large, small.

vue
<template>
  <div class="flex flex-wrap items-center mb-4">
    <el-button size="large">大号</el-button>
    <el-button>默认</el-button>
    <el-button size="small">小号</el-button>
    <el-button size="large" :icon="Search">搜索</el-button>
    <el-button :icon="Search">搜索</el-button>
    <el-button size="small" :icon="Search">搜索</el-button>
  </div>
  <div class="flex flex-wrap items-center mb-4">
    <el-button size="large" round>大号</el-button>
    <el-button round>默认</el-button>
    <el-button size="small" round>小号</el-button>
    <el-button size="large" :icon="Search" round>搜索</el-button>
    <el-button :icon="Search" round>搜索</el-button>
    <el-button size="small" :icon="Search" round>搜索</el-button>
  </div>
  <div class="flex flex-wrap items-center">
    <el-button :icon="Search" size="large" circle />
    <el-button :icon="Search" circle />

Custom Color beta ​

You can custom button color.

We will calculate hover color & active color automatically.

vue
<script lang="ts" setup>
import { isDark } from '~/composables/dark'

const brandColor = 'var(--el-color-primary)'
</script>

<template>
  <div>
    <el-button :color="brandColor" :dark="isDark">品牌按钮</el-button>
    <el-button :color="brandColor" :dark="isDark" plain>品牌朴素</el-button>

    <el-button :color="brandColor" :dark="isDark" disabled>禁用状态</el-button>
    <el-button :color="brandColor" :dark="isDark" disabled plain>
      禁用朴素
    </el-button>
  </div>
</template>
