# element-plus-mobile theme-chalk

> Theme styles bundled with `element-plus-mobile`.

## Installation

```shell
npm i element-plus-mobile
```

## Usage

Use Sass import

```css
@use 'element-plus-mobile/lib/theme-chalk/index.scss';
```

Or Use vite/webpack

```javascript
import 'element-plus-mobile/lib/theme-chalk/index.css'
```

Or

```html
<link
  rel="stylesheet"
  href="https://unpkg.com/element-plus-mobile/lib/theme-chalk/index.css"
/>
```

## Import on demand

```javascript
import 'element-plus-mobile/lib/theme-chalk/input.css'
import 'element-plus-mobile/lib/theme-chalk/select.css'

// ...
```
