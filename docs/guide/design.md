# 设计学科

<design-guide />

## 一致性

- **符合现实生活：**符合现实生活的流程和逻辑，
  并符合用户习惯的语言和习惯。

- **界面内一致：**所有元素应该一致，例如：
  设计风格、图标和文本、元素的位置等。

## 反馈

- **操作反馈：**让用户清晰感知自己的操作
  通过风格更新和互动效果。

- **视觉反馈：**通过更新或反映当前状态
  重新排列页面的元素。

## 效率

- **简化流程：**保持操作流程简单直观。

- **明确且清晰：**清楚地表达您的意图
  让用户能够快速理解并做出决策。

- **易于识别：**界面应该简单明了，
  这有助于用户识别并将他们从记忆和回忆中解放出来。

## 可控性

- **决策：** 可以接受有关运营的建议，但不要
  为用户做出决定。

- **受控后果：**应授予用户操作自由，
  包括取消、中止或终止当前操作。
