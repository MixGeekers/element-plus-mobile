# 无限滚动

到达页面底部时加载更多数据

## 基本用法

将 `v-infinite-scroll` 添加到列表中，以在滚动到底部时自动执行加载方法。

:::demo

infinite-scroll/basic

:::

## 禁用加载

:::demo

infinite-scroll/disable-loading

:::

## 指令

| 名称         | 描述                                                | 类型      | 默认 |
| ------------ | --------------------------------------------------- | --------- | ---- | --- |
| v-无限滚动   | 到达页面底部时加载更多数据                          | ^[功能]   | —    |
| 无限滚动禁用 | 已禁用                                              | ^[布尔值] | 假   |
| 无限滚动延迟 | 油门延迟（毫秒）                                    | ^[数字]   | 200  | 200 |
| 无限滚动距离 | 触发距离（px）                                      | ^[数字]   | 0    |
| 无限滚动立即 | 当初始状态无法填满内容时，是否立即执行loading方法。 | ^[布尔值] | 真实 |
