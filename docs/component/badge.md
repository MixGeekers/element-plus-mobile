# 徽章

按钮和图标上的数字或状态标记。

## 基本用法

显示新消息的数量。

:::demo 金额定义为接受数字或字符串的值。

badge/basic

:::

## 最大值

您可以自定义最大值。

:::demo 最大值由属性 max 定义，它是一个数字。请注意，它仅在 value 也是数字时才有效。

badge/max

:::

## 定制

显示数字以外的文本内容。或者您可以使用 `content` 插槽来自定义内容。

:::demo 当 value 为 String 时，可以显示自定义文本。或者使用 `content` 插槽。

badge/customize

:::

## 红点

用红点标记需要注意的内容。

:::demo 使用属性 `is-dot`。它是一个布尔值。

badge/dot

:::

## 偏移量 ^(2.7.0)

:::demo 设置徽章点的偏移量，格式为[left, top]，表示状态点相对于默认位置左侧和顶部的偏移量。

badge/offset

:::

## API

### 属性

| 名称              | 描述                                                | 类型                                                               | 默认   |
| ----------------- | --------------------------------------------------- | ------------------------------------------------------------------ | ------ | --- |
| 价值              | 显示值。                                            | ^[字符串] / ^[数字]                                                | ''     |
| 最大              | 最大值，超出时显示 `{max}+`。仅当值是数字时才有效。 | ^[数字]                                                            | 99     | 99  |
| 是点              | 如果显示一个小点。                                  | ^[布尔值]                                                          | 假     |
| 隐藏              | 隐藏徽章。                                          | ^[布尔值]                                                          | 假     |
| 类型              | 徽章类型。                                          | ^[枚举]`'primary' \| 'success' \| 'warning' \| 'danger' \| 'info'` | 危险   |
| 显示零 ^(2.6.0)   | 当值为零时是否显示徽章。                            | ^[布尔值]                                                          | 真实   |
| 颜色 ^(2.6.3)     | 点的背景颜色                                        | ^[字符串]                                                          |        |
| 偏移量 ^(2.7.0)   | 徽章偏移                                            | ^[数组]`[number, number]`                                          | [0, 0] |
| 徽章样式 ^(2.7.1) | 徽章定制风格                                        | ^[对象]`CSSProperties`                                             | —      |
| 徽章级 ^(2.7.1)   | 徽章定制类别                                        | ^[字符串]                                                          | —      |

### 老虎机

| 名称          | 描述           | 类型                       |
| ------------- | -------------- | -------------------------- |
| 默认          | 自定义默认内容 | -                          |
| 内容 ^(2.9.1) | 自定义徽章内容 | ^[对象]`{ value: string }` |
