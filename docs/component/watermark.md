# 水印

向页面添加特定文本或图案。

## 基本用法

最基本的用法。

:::demo

watermark/basic

:::

## 多行水印

使用`content`设置字符串数组来指定多行文本水印内容。

:::demo

watermark/multi-line

:::

## 图片水印

通过`image`指定图像地址。为保证图片高清且不拉伸，请设置宽度和高度，上传至少两倍于logo图片地址的宽度和高度。

:::demo

watermark/image

:::

## 自定义配置

通过配置自定义参数预览水印效果。

:::demo

watermark/custom

:::

## API

### 属性

| 名称   | 描述                                        | 类型                        | 默认                         |
| ------ | ------------------------------------------- | --------------------------- | ---------------------------- | --- |
| 宽度   | 水印的宽度，`content`的默认值是它自己的宽度 | ^[数字]                     | 120                          | 120 |
| 高度   | 水印的高度，`content`默认值为自身高度       | ^[数字]                     | 64                           | 64  |
| 旋转   | 绘制水印时，旋转角度，单位 `°`              | ^[数字]                     | -22                          |
| z 索引 | 附加水印元素的 z-index                      | ^[数字]                     | 9                            |
| 图像   | 图片来源，建议导出2x或3x图片，优先级高      | ^[字符串]                   | —                            |
| 内容   | 水印文字内容                                | ^[字符串]/^[数组]`string[]` | 元素加                       |
| 字体   | 文字样式                                    | [字体](#font)               | [字体](#font)                |
| 差距   | 水印之间的间距                              | ^[数组]`[number, number]`   | \[100, 100\]                 |
| 偏移   | 水印距容器左上角的偏移量。默认为`gap/2`     | ^[数组]`[number, number]`   | \[间隙\[0\]/2，间隙\[1\]/2\] |

### 字体

| 名称               | 描述     | 类型                                                                                 | 默认            |
| ------------------ | -------- | ------------------------------------------------------------------------------------ | --------------- | --------------- |
| 颜色               | 字体颜色 | ^[字符串]                                                                            | rgba(0,0,0,.15) | rgba(0,0,0,.15) |
| 字体大小           | 字体大小 | ^[数字] / ^[字符串]                                                                  | 16              | 16              |
| 字体粗细           | 字体粗细 | ^[枚举]`'normal' \| 'bold' \| 'lighter' \| 'bolder' \| number`                       | 正常            |
| 字体家族           | 字体家族 | ^[字符串]                                                                            | 无衬线          |
| 字体间隙 ^(2.11.5) | 字体间隙 | ^[数字]                                                                              | 3               |
| 字体样式           | 字体样式 | ^[枚举]`'none' \| 'normal' \| 'italic' \| 'oblique'`                                 | 正常            |
| 文本对齐           | 文本对齐 | ^[枚举]`'left' \| 'right' \| 'center' \| 'start' \| 'end' `                          | 中心            |
| 文本基线           | 文本基线 | ^[枚举]`'top' \| 'hanging' \| 'middle' \| 'alphabetic' \| 'ideographic' \| 'bottom'` | 悬挂            |

### 老虎机

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 添加水印的容器 |
