# 警报

显示重要的警报消息。

## 基本用法

警报组件是页面中不会自动消失的非覆盖元素。

:::demo Alert 提供 `type` 定义的 5 种主题，默认值为 `info`。 `primary` 已添加到 ^(2.9.11) 中。

alert/basic

:::

## 主题

Alert 提供两种不同的主题，`light` 和 `dark`。

:::demo 设置 `effect` 更改主题，默认为 `light`。

alert/theme

:::

## 可定制的关闭按钮

将关闭按钮自定义为文本或其他符号。

:::demo Alert 允许您配置它是否可关闭。关闭按钮文本和关闭回调也是可定制的。 `closable` 属性决定组件是否可以关闭。接受`boolean`，默认为`true`。您可以设置 `close-text` 属性来替换默认的十字符号作为关闭按钮。请注意，`close-text` 必须是字符串。 `close` 事件在组件关闭时触发。

alert/close-button

:::

## 带图标

显示图标可提高可读性。

:::demo 设置 `show-icon` 属性会显示与当前警报类型相对应的图标。或者使用`icon`插槽来自定义图标。

alert/icon

:::

## 居中文本

使用 `center` 属性使文本居中。

:::demo

alert/center

:::

## 带描述

描述包括包含更详细信息的消息。

:::demo 除了必需的 `title` 属性外，您还可以添加 `description` 属性来帮助您更详细地描述警报。描述只能存储文本字符串，并且会自动换行。

alert/description

:::

## 警报API

### 属性

| 名称     | 描述                 | 类型                                                                       | 默认 |
| -------- | -------------------- | -------------------------------------------------------------------------- | ---- |
| 标题     | 警报标题。           | ^[字符串]                                                                  | —    |
| 类型     | 警报类型。           | ^[枚举]`'primary' (2.9.11) \| 'success' \| 'warning' \| 'info' \| 'error'` | 信息 |
| 描述     | 描述性文字。         | ^[字符串]                                                                  | —    |
| 可关闭   | 是否可以解除警报。   | ^[布尔值]                                                                  | 真实 |
| 中心     | 内容是否放置在中心。 | ^[布尔值]                                                                  | 假   |
| 关闭文本 | 自定义关闭按钮文本。 | ^[字符串]                                                                  | —    |
| 显示图标 | 是否显示类型图标。   | ^[布尔值]                                                                  | 假   |
| 效果     | 主题风格。           | ^[枚举]`'light' \| 'dark'`                                                 | 光   |

### 活动

| 名称 | 描述             | 类型                                 |
| ---- | ---------------- | ------------------------------------ |
| 关闭 | 警报关闭时触发。 | ^[功能]`(event: MouseEvent) => void` |

### 老虎机

| 名称          | 描述             |
| ------------- | ---------------- |
| 默认          | 警报描述的内容。 |
| 标题          | 警报标题的内容。 |
| 图标 ^(2.9.7) | 警报图标的内容。 |
