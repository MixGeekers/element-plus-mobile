# DatePickerPanel ^（测试版）

`DatePickerPanel`是`DatePicker`的核心部件。

它尤其适合直接嵌入页面的场景，此时日历需要始终可见，而不是依赖输入框再弹出。

## 输入日期

基本日期选择器以“天”为单位。

:::demo

date-picker-panel/basic

:::

## 边框

默认情况下，日期选择器面板有边框，但在某些情况下您不想要它。
例如`DatePicker`不继承`border`。

:::demo

date-picker-panel/border

:::

## 禁用

`disabled` 属性确定日期选择器是否完全禁用。

:::demo

date-picker-panel/disabled

:::

## 类型

测量由 `type` 属性确定。

:::demo

date-picker-panel/all-types

:::

## 本地化

默认语言环境为英语，如果需要使用其他语言，请查看[国际化](/guide/i18n)

请注意，日期时间区域设置（月份名称、一周的第一天...）也在本地化中配置。

## API

### 属性

| 名称             | 描述                                                                                                   | 类型                                                                                                                                                           | 默认     |
| ---------------- | ------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| 模型值 / v 模型  | 绑定值，如果是 `range` 选择器，则数组长度应为 2                                                        | ^[数字] / ^[字符串] / ^[日期] / ^[数组]`number[] \| string[] \| Date[]`                                                                                        | ''       |
| 边框             | 日期选择器是否有边框                                                                                   | ^[布尔值]                                                                                                                                                      | 真实     |
| 已禁用           | DatePicker 是否被禁用                                                                                  | ^[布尔值]                                                                                                                                                      | 假       |
| 可清除           | 是否显示清除按钮                                                                                       | ^[布尔值]                                                                                                                                                      | 真实     |
| 可编辑 ^(2.13.0) | 输入是否可编辑                                                                                         | ^[布尔值]                                                                                                                                                      | 真实     |
| 类型             | 选取器的类型                                                                                           | ^[枚举]`'year' \| 'years' \|'month' \| 'months' \| 'date' \| 'dates' \| 'datetime' \| 'week' \| 'datetimerange' \| 'daterange' \| 'monthrange' \| 'yearrange'` | 日期     |
| 默认值           | 可选，日历的默认日期                                                                                   | ^[对象]`Date \| [Date, Date]`                                                                                                                                  | —        |
| 默认时间         | 可选，选择日期范围时使用的时间值                                                                       | ^[对象]`Date \| [Date, Date]`                                                                                                                                  | —        |
| 值格式           | 可选，绑定值的格式。如果未指定，绑定值将为 Date 对象                                                   | ^[字符串]                                                                                                                                                      | —        |
| 日期格式         | 可选，输入内部面板中显示的日期格式                                                                     | ^[字符串] 请参阅[日期格式](https://day.js.org/docs/en/display/format)                                                                                          | 年-月-日 |
| 时间格式         | 可选，输入内部面板中显示的时间格式                                                                     | ^[字符串] 请参阅[日期格式](https://day.js.org/docs/en/display/format)                                                                                          | 时:分:秒 |
| 取消链接面板     | 取消范围选择器中两个日期面板的链接                                                                     | ^[布尔值]                                                                                                                                                      | 假       |
| 禁用日期         | 确定某个日期是否被禁用并以该日期作为参数的函数。应该返回一个布尔值                                     | ^[功能]`(data: Date) => boolean`                                                                                                                               | —        |
| 快捷方式         | 用于设置快捷方式选项的对象数组                                                                         | ^[数组]`Array<{ text: string, value: Date \| Function }>`                                                                                                      | []       |
| 细胞类别名称     | 设置自定义类名                                                                                         | ^[功能]`(data: Date) => string`                                                                                                                                | —        |
| 显示页脚         | 是否显示日期选择器为 1 的页脚 ^[enum]`'dates' \| 'months' \| 'years' \| 'datetime' \| 'datetimerange'` | ^[布尔值]                                                                                                                                                      | 假       |
| 显示确认         | 是否显示确认按钮                                                                                       | ^[布尔值]                                                                                                                                                      | 假       |
| 显示周数         | 除星期外还显示星期数                                                                                   | ^[布尔值]                                                                                                                                                      | 假       |

### 活动

| 名称           | 描述                                        | 类型                                                                                  |
| -------------- | ------------------------------------------- | ------------------------------------------------------------------------------------- |
| 日历更改       | 当日历选择的日期更改时触发。仅适用于`range` | ^[功能]`(val: [Date, null \| Date]) => void`                                          |
| 面板更换       | 单击导航按钮时触发。                        | ^[功能]`(date: Date \| [Date, Date], mode: 'month' \| 'year', view?: string) => void` |
| 清除 ^(2.13.1) | 单击清除按钮时触发                          | ^[功能]`() => void`                                                                   |
