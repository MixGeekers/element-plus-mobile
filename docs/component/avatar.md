# 头像

化身可以用来代表人或物体。它支持图像、图标或字符。

## 基本用法

使用 `shape` 和 `size` 道具设置头像的形状和大小。

:::demo

avatar/basic

:::

## 类型

它支持图像、图标或字符。

:::demo

avatar/types

:::

## 头像 API

### 头像属性

| 名称 | 描述                                     | 类型                                                              | 默认 |
| ---- | ---------------------------------------- | ----------------------------------------------------------------- | ---- |
| 图标 | 图标的表示类型，有关图标组件的更多信息。 | ^[字符串] / ^[组件]                                               | —    |
| 尺寸 | 头像尺寸。                               | ^[数字] / ^[枚举]`'large' \| 'default' \| 'small'`                | —    |
| 形状 | 头像形状。                               | ^[枚举]`'circle' \| 'square'`                                     | —    |
| 来源 | 图像头像的图像来源。                     | `string`                                                          | —    |
| 源集 | 图像头像的原生属性 `srcset`。            | `string`                                                          | —    |
| 替代 | 图像头像的原生属性 `alt`。               | `string`                                                          | —    |
| 适合 | 设置图像如何适合图像头像的容器。         | ^[枚举]`'fill' \| 'contain' \| 'cover' \| 'none' \| 'scale-down'` | 封面 |

### 头像活动

| 名称 | 描述                   | 类型                        |
| ---- | ---------------------- | --------------------------- |
| 错误 | 当图像加载错误时触发。 | ^[功能]`(e: Event) => void` |

### 头像槽

| 名称 | 描述             |
| ---- | ---------------- |
| 默认 | 自定义头像内容。 |
