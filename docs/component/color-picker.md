# 颜色选择器

ColorPicker是一个支持多种颜色格式的颜色选择器。

触发器保持紧凑，颜色面板会以底部面板展开。

## 基本用法

:::demo ColorPicker 需要将字符串类型变量绑定到 v-model。

color-picker/basic

:::

## 阿尔法

:::demo ColorPicker 支持 Alpha 通道选择。要激活 Alpha 选择，只需添加 `show-alpha` 属性。

color-picker/alpha

:::

## 预定义颜色

:::demo ColorPicker 支持预定义颜色选项

color-picker/predefined-color

:::

## 尺寸

:::demo

color-picker/sizes

:::

## API

### 属性

|名称 |描述 |类型 |默认 |
| ------------------------ | | -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- |
|模型值 / v 模型 |绑定值| ^[字符串] | — |
|已禁用 |是否禁用ColorPicker | ^[布尔值] |假 |
|可清除 ^(2.13.1) |是否显示清除按钮| ^[布尔值] |真实|
|尺寸|颜色选择器的大小 | ^[枚举]`'large' \| 'default' \| 'small'` | — |
|显示阿尔法 |是否显示 alpha 滑块 | ^[布尔值] |假 |
|颜色格式 | v-model 的颜色格式 | ^[枚举]`'rgb' \| 'prgb' \| 'hex' \| 'hex3' \| 'hex4' \| 'hex6' \| 'hex8' \| 'name' \| 'hsl' \| 'hsv'` | ^[枚举]`'hex' (when show-alpha is false) \| 'rgb' (when show-alpha is true)` |
|波普尔级| ColorPicker 下拉菜单的自定义类名 | ^[字符串] / ^[对象] | ''|
|波普风格 ^(2.11.4) | ColorPicker 下拉菜单的自定义样式 | ^[字符串] / ^[对象] | — |
|预定义 |预定义颜色选项 | ^[数组]`string[]` | — |
|验证事件 |是否触发表单验证 | ^[布尔值] |真实|
|标签索引 |颜色选择器选项卡索引 | ^[字符串] / ^[数字] | 0 |
|咏叹调标签 ^(a11y) ^(2.7.2) | ColorPicker 咏叹调标签 | ^[字符串] | — |
|空值 ^(2.10.3) |组件的空值，[参见配置提供程序](./config-provider.md#empty-values-configurations) | ^[数组] | — |
|清除值 ^(2.10.3) |清除返回值，[参见配置提供程序](./config-provider.md#empty-values-configurations) | ^[字符串] / ^[数字] / ^[布尔值] / ^[函数] | — |
|编号 |颜色选择器 ID | ^[字符串] | — |
|标签 ^(a11y) ^(已弃用) | ColorPicker 咏叹调标签 | ^[字符串] | — |
|持久 ^(2.10.5) |当颜色选择器 inactive and persist 为 false 时，颜色面板将被破坏 | ^[布尔值] |真实|

### 活动

| 名称           | 描述                     | 类型                                 |
| -------------- | ------------------------ | ------------------------------------ |
| 改变           | 当输入值改变时触发       | ^[功能]`(value: string) => void`     |
| 主动改变       | 当当前活动颜色改变时触发 | ^[功能]`(value: string) => void`     |
| 焦点 ^(2.4.0)  | 组件聚焦时触发           | ^[功能]`(event: FocusEvent) => void` |
| 模糊 ^(2.4.0)  | 当组件模糊时触发         | ^[功能]`(event: FocusEvent) => void` |
| 清除 ^(2.13.1) | 单击清除按钮时触发       | ^[功能]`() => void`                  |

### 暴露

| 名称           | 描述                 | 类型                |
| -------------- | -------------------- | ------------------- |
| 颜色           | 当前颜色对象         | ^[对象]`Color`      |
| 显示 ^(2.3.3)  | 手动显示 ColorPicker | ^[功能]`() => void` |
| 隐藏 ^(2.3.3)  | 手动隐藏 ColorPicker | ^[功能]`() => void` |
| 焦点 ^(2.3.13) | 聚焦选择器元素       | ^[功能]`() => void` |
| 模糊 ^(2.3.13) | 模糊选择器元素       | ^[功能]`() => void` |
