# 选择

当有很多选项时，使用下拉菜单来显示并选择所需的选项。

:::tip

版本^(2.5.0)之后，`el-select`的默认宽度更改为`100%`。当以内联形式使用时，宽度会折叠。为了正确显示宽度，您需要给 `el-select` 一个特定的宽度（例如：[示例](https://github.com/element-plus/element-plus/issues/15834#issuecomment-1936919229)）。

:::

## 选项过滤

您可以过滤您想要的选项。

:::demo 将 `filterable` 添加到 `el-select` 即可启用过滤。默认情况下，Select 将查找 `label` 属性包含输入值的所有选项。如果您更喜欢其他过滤策略，可以传递`filter-method`。 `filter-method`是当输入值改变时调用的`Function`，其参数为当前输入值。

### 标签工具提示 ^(2.13.3)

:::tip 回退机制

tag-tooltip 中的属性遵循以下优先顺序：

1. 标签工具提示对象中显式定义的字段。
2. 从 el-select 继承的共享属性（例如，effect、popper-class、popper-style、teleported、append-to、popper-options）。3.底层el-tooltip组件的默认值。
   这允许您覆盖标签的特定工具提示行为，同时保持与默认情况下的“选择”下拉列表的一致性。

:::

:::tip 定制集装箱定位

将工具提示附加到自定义容器时（通过 `append-to` 属性），容器应配置为 `position: relative` 或 `position: absolute` 以确保准确定位。此外，如果需要防止工具提示溢出其边界，您可以将 `overflow: hidden` 应用于容器。

:::
