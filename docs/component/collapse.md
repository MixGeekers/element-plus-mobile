# 折叠

使用 Collapse 来存储内容。

## 基本用法

您可以展开多个面板

:::demo

collapse/basic

:::

## 手风琴

手风琴模式下，一次只能展开一个面板

:::demo 使用 `accordion` 属性激活手风琴模式。

collapse/accordion

:::

## 自定义标题

除了使用 `title` 属性之外，您还可以使用命名槽自定义面板标题，这使得添加自定义内容，例如图标，可能。

:::tip

从版本 ^(2.9.10) 开始，`title` 插槽提供了 `isActive` 属性，指示当前折叠项是否处于活动状态。

:::

:::demo

collapse/customization

:::

## 自定义图标 ^(2.8.3)

除了使用 `icon` 属性之外，您还可以使用命名槽自定义面板项的图标，这使得添加自定义内容。

:::demo

collapse/custom-icon

:::

## 自定义图标位置^(2.9.10)

使用 `expand-icon-position` 属性，您可以自定义图标位置。

:::demo

collapse/custom-icon-position

:::

## 防止崩溃 ^(2.9.11)

设置`before-collapse`属性，如果返回`false`或者返回`Promise`然后被拒绝，将停止折叠。

:::demo

collapse/prevent-collapsing

:::

## 折叠 API

### 折叠属性

| 名称                   | 描述                                                                                             | 类型                                       | 默认 |
| ---------------------- | ------------------------------------------------------------------------------------------------ | ------------------------------------------ | ---- |
| 模型值 / v 模型        | 当前活动面板，手风琴模式下类型为 `string`，否则为 `array`                                        | ^[字符串] / ^[数组]                        | []   |
| 手风琴                 | 是否激活手风琴模式                                                                               | ^[布尔值]                                  | 假   |
| 展开图标位置 ^(2.9.10) | 设置展开图标位置                                                                                 | ^[枚举]`'left' \| 'right' `                | 对   |
| 崩溃前 ^(2.9.11)       | before-collapse 钩子在折叠状态改变之前。如果返回 `false` 或返回 `Promise` 然后被拒绝，将停止折叠 | ^[功能]`() => Promise<boolean> \| boolean` | —    |

### 崩溃事件

| 名称 | 描述                                                                  | 类型                                            |
| ---- | --------------------------------------------------------------------- | ----------------------------------------------- |
| 改变 | 当活动面板改变时触发，手风琴模式下参数类型为 `string`，否则为 `array` | ^[功能]`(activeNames: array \| string) => void` |

### 折叠槽

| 名称 | 描述           | 子标签   |
| ---- | -------------- | -------- |
| 默认 | 自定义默认内容 | 折叠项目 |

### 崩溃暴露

| 名称         | 描述               | 类型                                                 |
| ------------ | ------------------ | ---------------------------------------------------- |
| 活动名称     | 当前活动的面板名称 | ^[对象]`ComputedRef<(string \| number)[]>`           |
| 设置活动名称 | 设置活动面板名称   | ^[功能]`(activeNames: (string \| number)[]) => void` |

## 折叠项目 API

### 折叠项目属性

| 名称          | 描述           | 类型                | 默认     |
| ------------- | -------------- | ------------------- | -------- |
| 名称          | 面板唯一标识   | ^[字符串] / ^[数字] | —        |
| 标题          | 小组标题       | ^[字符串]           | ''       |
| 图标 ^(2.8.3) | 折叠项目的图标 | ^[字符串] / ^[组件] | 向右箭头 |
| 已禁用        | 禁用折叠项     | ^[布尔值]           | 假       |

### 折叠物品槽

| 名称          | 描述               | 类型                                 |
| ------------- | ------------------ | ------------------------------------ |
| 默认          | 折叠项目的内容     | —                                    |
| 标题          | 折叠项目标题的内容 | ^[对象]`{ isActive: boolean }`       |
| 图标 ^(2.8.3) | 折叠项目图标       | 的内容^[对象]`{ isActive: boolean }` |

### 折叠项目暴露

| 名称         | 描述                       | 类型                                       |
| ------------ | -------------------------- | ------------------------------------------ |
| 处于活动状态 | 当前折叠项是否处于活动状态 | ^[对象]`ComputedRef<boolean \| undefined>` |
