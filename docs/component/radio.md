# 电台

多个选项中的单一选择。

:::warning

`label` 与 `value` 已被**弃用**，`label` 仅用作显示文本，此操作\*\*将在 ^(3.0.0) 中删除，考虑切换到新 API。

:::

:::tip

^(2.6.0)中添加了新的API `value`，文档中的示例均使用`value`。
如果您使用的版本**低于**^(2.6.0)，请参考：

:::

```vue
<template>
  <el-radio-group v-model="radio1">
    <!-- works when >=2.6.0, recommended ✔️ not work when <2.6.0 ❌ -->
    <el-radio value="Value 1">Option 1</el-radio>
    <!-- works when <2.6.0, deprecated act as value when >=3.0.0 -->
    <el-radio label="Label 2 & Value 2">Option 2</el-radio>
  </el-radio-group>
</template>
```

## 基本用法

收音机不应该有太多的选择。否则，请改用 Select 组件。

:::demo 创建无线电组件很简单，只需将变量绑定到无线电的 `v-model` 即可。它等于所选无线电的 `value` 的值。 `value` 的类型为 `String`、`Number` 或 `Boolean`。

:::

## 无线电 API

### 无线电属性

| 名称            | 描述                                                | 类型                                     | 默认 |
| --------------- | --------------------------------------------------- | ---------------------------------------- | ---- |
| 模型值 / v 模型 | 绑定值                                              | ^[字符串] / ^[数字] / ^[布尔值]          | —    |
| 值 ^(2.6.0)     | 广播的价值                                          | ^[字符串] / ^[数字] / ^[布尔值]          | —    |
| 标签            | 电台的标签。如果没有`value`，则`label`将充当`value` | ^[字符串] / ^[数字] / ^[布尔值]          | —    |
| 已禁用          | 无线电是否被禁用                                    | ^[布尔值]                                | 假   |
| 边框            | 是否在Radio周围添加边框                             | ^[布尔值]                                | 假   |
| 尺寸            | 收音机的尺寸                                        | ^[枚举]`'large' \| 'default' \| 'small'` | —    |
| 名称            | 原生 `name` 属性                                    | ^[字符串]                                | —    |

### 电台活动

| 名称 | 描述               | 类型                                                  |
| ---- | ------------------ | ----------------------------------------------------- |
| 改变 | 当绑定值改变时触发 | ^[功能]`(value: string \| number \| boolean) => void` |

### 无线电老虎机

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
