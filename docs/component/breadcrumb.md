# 面包屑导航

显示当前页面的位置，以便更轻松地返回浏览。

## 基本用法

:::demo 在`el-breadcrumb`中，每个`el-breadcrumb-item`都是一个标签，代表从主页开始的每个级别。该组件具有 `String` 属性 `separator`，它确定分隔符。它的默认值为“/”。

breadcrumb/basic

:::

## 图标分隔符

:::demo 设置`separator-icon`使用`svg icon`作为分隔符，它将覆盖`separator`

breadcrumb/icon

:::

## 面包屑 API

### 面包屑属性

| 名称       | 描述                 | 类型                | 默认 |
| ---------- | -------------------- | ------------------- | ---- |
| 分隔符     | 分隔符               | ^[字符串]           | /    |
| 分隔符图标 | 图标分隔符的图标组件 | ^[字符串] / ^[组件] | —    |

### 面包屑槽

| 名称 | 描述           | 子标签     |
| ---- | -------------- | ---------- |
| 默认 | 自定义默认内容 | 面包屑项目 |

## 面包屑条目 API

### BreadcrumbItem 属性

| 名称 | 描述                                     | 类型                                  | 默认 |
| ---- | ---------------------------------------- | ------------------------------------- | ---- |
| 至   | 链路的目标路由，与`vue-router`的`to`相同 | ^[字符串] / ^[对象]`RouteLocationRaw` | ''   |
| 替换 | 如果是`true`，则导航不会留下历史记录     | ^[布尔值]                             | 假   |

### 面包屑项目槽

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
