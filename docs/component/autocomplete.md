## API

### 属性

| 名称                              | 描述                                                                              | 类型                                                                                     | 默认     |
| --------------------------------- | --------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- | -------- |
| 模型值 / v 模型                   | 绑定值                                                                            | ^[字符串]                                                                                | —        |
| 占位符                            | 自动完成                                                                          | 的占位符^[字符串]                                                                        | —        |
| 可清除                            | 是否显示清除按钮                                                                  | ^[布尔值]                                                                                | 假       |
| 已禁用                            | 是否禁用自动完成                                                                  | ^[布尔值]                                                                                | 假       |
| 值键                              | 用于显示的输入建议对象的键名                                                      | ^[字符串]                                                                                | 价值     |
| 去抖                              | 打字时的去抖延迟（以毫秒为单位） ^[数字]                                          | 300                                                                                      | 300      |
| 安置                              | 弹出菜单的位置                                                                    | ^[枚举]`'top' \| 'top-start' \| 'top-end' \| 'bottom' \| 'bottom-start' \| 'bottom-end'` | 底部开始 |
| 获取建议                          | 获取输入建议的方法。当建议准备就绪时，调用 `callback(data:[])` 将其返回到自动完成 | ^[数组] / ^[函数]`(queryString: string, callback: callbackfn) => void`                   | —        |
| 焦点触发                          | 输入焦点时是否显示建议                                                            | ^[布尔值]                                                                                | 真实     |
| 不匹配时选择                      | 当没有自动完成匹配时是否在输入时发出 `select` 事件                                | ^[布尔值]                                                                                | 假       |
| 名称                              | 与本机输入中的 `name` 相同                                                        | ^[字符串]                                                                                | —        |
| 咏叹调标签 ^(a11y) ^(2.7.2)       | 原生 `aria-label` 属性                                                            | ^[字符串]                                                                                | —        |
| 隐藏加载                          | 远程搜索时是否隐藏加载图标                                                        | ^[布尔值]                                                                                | 假       |
| 波普尔级                          | 自动完成下拉菜单的自定义类名称                                                    | ^[字符串] / ^[对象]                                                                      | ''       |
| 波普风格 ^(2.11.4)                | 自动完成下拉菜单的自定义样式                                                      | ^[字符串] / ^[对象]                                                                      | —        |
| 传送                              | 选择下拉列表是否传送到正文                                                        | ^[布尔值]                                                                                | 真实     |
| 追加到 ^(2.9.9)                   | 其中选择下拉列表附加到                                                            | ^[CSS选择器] / ^[HTMLElement]                                                            | —        |
| 突出显示第一项                    | 是否默认突出显示远程搜索建议中的第一项                                            | ^[布尔值]                                                                                | 假       |
| 适合输入宽度                      | 下拉列表的宽度是否与输入的宽度相同                                                | ^[布尔值]                                                                                | 假       |
| popper-append-to-body ^（已弃用） | 是否将下拉列表附加到正文。如果下拉菜单的位置错误，可以尝试将此属性设置为 false    | ^[布尔值]                                                                                | 假       |
| 循环导航 ^(2.11.4)                | 键盘导航是否从结束到开始循环^[布尔值]                                             | 真实                                                                                     |
| [输入道具](./input.md#attributes) | —                                                                                 | —                                                                                        | —        |

### 活动

| 名称 | 描述                           | 类型                                         |
| ---- | ------------------------------ | -------------------------------------------- |
| 模糊 | 当输入模糊时触发               | ^[功能]`(event: FocusEvent) => void`         |
| 焦点 | 当输入聚焦时触发               | ^[功能]`(event: FocusEvent) => void`         |
| 输入 | 当输入值改变时触发             | ^[功能]`(value: string \| number) => void`   |
| 清除 | 单击清除按钮清除输入时触发     | ^[功能]`() => void`                          |
| 选择 | 单击建议时触发                 | ^[功能]`(item: Record<string, any>) => void` |
| 改变 | 当输入值内的图标发生变化时触发 | ^[功能]`(value: string \| number) => void`   |

### 老虎机

| 名称           | 描述                 | 类型                                   |
| -------------- | -------------------- | -------------------------------------- |
| 默认           | 输入建议的自定义内容 | ^[对象]`{ item: Record<string, any> }` |
| 标头 ^(2.10.6) | 下拉菜单顶部的内容   | -                                      |
| 页脚 ^(2.10.6) | 下拉菜单底部的内容   | -                                      |
| 前缀           | 内容作为输入前缀     | -                                      |
| 后缀           | 内容作为输入后缀     | -                                      |
| 前置           | 在输入之前添加的内容 | -                                      |
| 附加           | 输入后附加的内容     | -                                      |
| 加载 ^(2.5.0)  | 覆盖加载内容         | -                                      |

### 暴露

| 名称              | 描述                     | 类型                                            |
| ----------------- | ------------------------ | ----------------------------------------------- |
| 激活              | 如果自动完成功能已激活   | ^[对象]`Ref<boolean>`                           |
| 模糊              | 模糊输入元素             | ^[功能]`() => void`                             |
| 关闭              | 折叠建议列表             | ^[功能]`() => void`                             |
| 焦点              | 聚焦输入元素             | ^[功能]`() => void`                             |
| 处理选择          | 单击建议时触发           | ^[功能]`(item: any) => promise<void>`           |
| 处理按键输入      | 处理键盘输入事件         | ^[功能]`() => promise<void>`                    |
| 突出显示索引      | 当前突出显示的项目的索引 | ^[对象]`Ref<number>`                            |
| 突出              | 突出显示建议中的一项     | ^[功能]`(itemIndex: number) => void`            |
| 输入参考          | el-input 组件实例        | ^[对象]`Ref<ElInputInstance>`                   |
| 加载              | 远程搜索加载指示器       | ^[对象]`Ref<boolean>`                           |
| 波普尔参考        | el-tooltip 组件实例      | ^[对象]`Ref<ElTooltipInstance>`                 |
| 建议              | 获取建议结果             | ^[对象]`Ref<record<string, any>[]>`             |
| 获取数据 ^(2.8.4) | 加载建议列表             | ^[功能]`(queryString: string) => promise<void>` |
