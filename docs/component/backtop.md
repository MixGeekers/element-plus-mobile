# 后台

返回顶部的按钮。

## 基本用法

向下滚动以查看右下角的按钮。

:::demo

backtop/basic

:::

## 定制

显示区域为 40px \* 40px。

:::demo

backtop/custom

:::

## API

### 属性

| 名称       | 描述                                   | 类型      | 默认 |
| ---------- | -------------------------------------- | --------- | ---- | --- |
| 目标       | 触发滚动的目标。                       | ^[字符串] | —    |
| 能见度高度 | 在滚动高度达到该值之前，按钮不会显示。 | ^[数字]   | 200  | 200 |
| 对         | 正确的距离。                           | ^[数字]   | 40   |
| 底部       | 底部距离。                             | ^[数字]   | 40   |

### 活动

| 名称 | 描述         | 参数                               |
| ---- | ------------ | ---------------------------------- |
| 点击 | 单击时触发。 | ^[功能]`(evt: MouseEvent) => void` |

### 老虎机

| 名称 | 描述             |
| ---- | ---------------- |
| 默认 | 自定义默认内容。 |
