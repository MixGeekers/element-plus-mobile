# 下拉菜单

用于显示链接和操作列表的可切换菜单。

## 基本用法

将鼠标悬停在下拉菜单上可将其展开以执行更多操作。

:::demo 触发元素由默认的`slot`渲染，下拉部分由名为`dropdown`的`slot`渲染。默认情况下，当您将鼠标悬停在触发元素上而无需单击它时，会显示下拉列表。

dropdown/basic-usage

:::

## 安置

支持6个展示位置。

:::demo 设置 `placement` 属性以使下拉菜单出现在不同的位置。

dropdown/placements

:::

## 触发元素

使用按钮触发下拉列表。

:::demo 使用`split-button` 将触发元素拆分为按钮组，左按钮为普通按钮，右按钮为实际触发目标。如果您想在第 3 项和第 4 项之间插入分隔线，只需将 `divided` 属性添加到第 4 项即可。

dropdown/triggering-element

:::

## 如何触发

单击触发元素或将鼠标悬停在其上。

:::demo 使用属性 `trigger`。默认为`hover`。

dropdown/how-to-trigger

:::

## 菜单隐藏行为

使用 `hide-on-click` 定义单击时菜单是否关闭。

:::demo 默认情况下，单击菜单项时菜单将关闭，可以通过将 hide-on-click 设置为 false 来关闭它。

dropdown/menu-hiding-behavior

:::

## 命令事件

单击每个下拉项会触发一个事件，其参数由每个项分配。

:::demo

dropdown/command-event

:::

## 下拉方法

您可以通过手动使用 `handleOpen` 或 `handleClose` 打开或关闭下拉菜单

:::demo

dropdown/dropdown-methods

:::

## 尺寸

除了默认大小外，Dropdown 组件还提供了三种附加大小供您在不同场景中进行选择。

:::demo 使用属性 `size` 通过 `large`、`default` 或 `small` 设置附加尺寸。

dropdown/sizes

:::

## 虚拟触发^(2.11.3)

有时我们想在其他触发元素上渲染下拉菜单，我们可以将触发器和内容分开。

### 下拉槽

| 名称     | 描述                                                                                                                  | 子标签   |
| -------- | --------------------------------------------------------------------------------------------------------------------- | -------- |
| 默认     | 下拉菜单的内容。注意：必须是有效的 html dom 元素（例如 `<span>, <button> etc.`）或 `el-component`，才能附加触发侦听器 | —        |
| 下拉菜单 | 下拉菜单的内容，通常是 `<el-dropdown-menu>` 元素                                                                      | 下拉菜单 |

### 下拉事件

| 名称     | 描述                                                   | 类型                              |
| -------- | ------------------------------------------------------ | --------------------------------- |
| 点击     | 如果`split-button`是`true`，则点击左键时触发           | ^[功能]`(e: MouseEvent) => void`  |
| 命令     | 单击下拉项时触发，参数是从下拉项派发的命令             | ^[功能]`(...args: any[]) => void` |
| 可见变化 | 当下拉出现/消失时触发，出现时参数为 true，否则为 false | ^[功能]`(val: boolean) => void`   |

### 下拉菜单暴露

| 方法     | 描述         | 类型                |
| -------- | ------------ | ------------------- |
| 处理打开 | 打开下拉菜单 | ^[功能]`() => void` |
| 处理关闭 | 关闭下拉菜单 | ^[功能]`() => void` |

## 下拉项 API

### 下拉菜单项属性

| 名称   | 描述                                    | 类型                          | 默认 |
| ------ | --------------------------------------- | ----------------------------- | ---- |
