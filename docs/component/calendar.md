# 日历

显示日期。

## 基础

:::demo 设置 `value` 以指定当前显示的月份。如果未指定`value`，则显示当前月份。 `value`支持双向绑定。

calendar/basic

:::

## 控制器类型 ^(2.13.1)

:::demo 您可以设置日历标题的控制器类型。设置`select`时，可以使用`formatter`自定义`label`。

calendar/controller-type

:::

## 自定义内容

:::demo 通过将 `scoped-slot` 设置为 `date-cell` 来自定义日历单元格中显示的内容。在`scoped-slot`中可以获取日期（当前单元格的日期）、数据（包括type、isSelected、day属性）。详细信息请参考下面的API文档。

calendar/customize

:::

## 范围

:::demo 设置 `range` 属性以指定日历的显示范围。开始时间必须是星期一，结束时间必须是星期日，时间跨度不能超过两个月。

calendar/range

:::

## 自定义标题

:::demo

calendar/header

:::

## 本地化

默认语言环境为英语，如果需要使用其他语言，请查看[国际化](/guide/i18n)

请注意，日期时间区域设置（月份名称、一周的第一天...）也在本地化中配置。

## API

### 属性

| 名称                 | 描述                                                                                                                   | 类型                                                                  | 默认 |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- | ---- |
| 模型值 / v 模型      | 绑定值                                                                                                                 | ^[日期]                                                               | —    |
| 范围                 | 时间范围，包括开始时间和结束时间。开始时间必须是一周的开始日期，结束时间必须是一周的结束日期，时间跨度不能超过两个月。 | ^[数组]`[Date, Date]`                                                 | —    |
| 控制器类型 ^(2.13.1) | 日历标题控制器的类型                                                                                                   | ^[枚举]`'button' \| 'select'`                                         | 按钮 |
| 格式化程序 ^(2.13.1) | 当 `controller-type` 为“选择”时格式化标签                                                                              | ^[功能]`(value: number, type: 'year' \| 'month') => string \| number` | —    |

### 老虎机

| 名称       | 描述                                                                                                                                                    | 类型                                                                                                                       |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| 日期单元格 | `type` 表示日期所属月份，可选值为上月、当月、下月； `isSelected`表示是否选择日期； `day` 是格式为 `YYYY-MM-DD` 的格式化日期； `date` 是单元格代表的日期 | ^[对象]`{ data: { type: 'prev-month' \| 'current-month' \| 'next-month', isSelected: boolean, day: string, date: Date } }` |
| 标题       | 日历标题的内容                                                                                                                                          | ^[对象]`{ date: string }`                                                                                                  |

### 暴露

|名称 |描述 |类型 |
| ------------------------ | | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
|选定日期 |当前选择的日期 | ^[对象]`ComputedRef<Dayjs \| undefined>` |
|挑选日 |选择特定日期 | ^[功能]`(day: dayjs.Dayjs) => void` |
|选择日期 |选择日期 | ^[功能]`(type: CalendarDateType) => void` |
|计算验证日期范围 |根据开始和结束日期计算有效日期范围 | ^[功能]`(startDayjs: dayjs.Dayjs, endDayjs: dayjs.Dayjs) => [dayjs.Dayjs, dayjs.Dayjs][]` |
