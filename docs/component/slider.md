# 滑块

在固定范围内拖动滑块。

## 基本用法

拖动滑块时会显示当前值。

:::demo 通过设置绑定值来自定义滑块的初始值。

slider/basic-usage

:::

## 离散值

这些选项可以是离散的。

:::demo 使用 `step` 属性设置步长。您可以通过设置 `show-stops` 属性来显示断点。

slider/discrete-values

:::

## 带输入框的滑块

通过输入框设置值。

:::demo 设置`show-input`属性以在右侧显示输入框。

slider/slider-with-input-box

:::

## 尺寸

:::demo

slider/sizes

:::

## 安置

您可以自定义工具提示的位置。

:::demo

slider/placement

:::

## 范围选择

支持选择一个值范围。

:::demo 设置 `range` 属性会激活范围模式，其中绑定值是由两个边界值组成的数组。

## 垂直模式

:::demo 将 `vertical` 属性设置为 `true` 启用垂直模式。在垂直模式下，需要 `height` 属性。

## 显示标记

:::demo 设置此 `marks` 属性可以在滑块上显示标记。

## 限制值 ^(2.13.6)

:::demo 设置 `step="mark"` 将滑块值限制为标记。

### 活动

| 名称 | 描述                                                       | 类型                                           |
| ---- | ---------------------------------------------------------- | ---------------------------------------------- |
| 改变 | 当值改变时触发（如果鼠标被拖动，该事件仅在释放鼠标时触发） | ^[功能]`(value: Arrayable<number>) => boolean` |
| 输入 | 数据变化时触发（滑动时实时发出）                           | ^[功能]`(value: Arrayable<number>) => boolean` |
