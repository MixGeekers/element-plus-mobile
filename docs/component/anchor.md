## 锚点 API

### 锚属性

| 物业                  | 描述                                   | 类型                                  | 默认       |
| --------------------- | -------------------------------------- | ------------------------------------- | ---------- | --- |
| 集装箱                | 滚动容器。                             | `string` \| `HTMLElement` \| `Window` | —          |
| 偏移                  | 设置锚点滚动的偏移量。                 | `number`                              | 0          |
| 绑定                  | 开始触发锚点的元素的偏移量。           | `number`                              | 15         | 15  |
| 持续时间              | 设置容器的滚动持续时间，以毫秒为单位。 | `number`                              | 300        | 300 |
| 标记                  | 是否显示标记。                         | ^[布尔值]                             | 真实       |
| 类型                  | 设置锚点类型。                         | ^[枚举]`'default' \| 'underline'`     | `default`  |
| 方向                  | 设置锚点方向。                         | ^[枚举]`'vertical' \| 'horizontal'`   | `vertical` |
| 选择滚动顶部 ^(2.9.2) | 滚动是否选择顶部链接                   | ^[布尔值]                             | 假         |

### 主播活动

| 名称 | 描述                 | 类型                                            |
| ---- | -------------------- | ----------------------------------------------- |
| 改变 | 步骤改变时的回调     | ^[功能]`(href: string) => void`                 |
| 点击 | 当用户点击链接时触发 | ^[功能]`(e: MouseEvent, href?: string) => void` |

### 主播曝光

| 名称   | 描述                 | 类型                            |
| ------ | -------------------- | ------------------------------- |
| 滚动至 | 手动滚动到特定位置。 | ^[功能]`(href: string) => void` |

### 锚槽

| 名称 | 描述                |
| ---- | ------------------- |
| 默认 | AnchorLink 组件列表 |

### AnchorLink 属性

| 物业 | 描述               | 类型     | 默认 |
| ---- | ------------------ | -------- | ---- |
| 标题 | 锚链接的文本内容。 | `string` | —    |
| 链接 | 锚链接的地址。     | `string` | —    |

### AnchorLink 插槽

| 名称   | 描述           |
| ------ | -------------- |
| 默认   | 锚链接的内容。 |
| 子链接 | 子链接的插槽。 |
