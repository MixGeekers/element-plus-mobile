# 容器

容器组件用于脚手架页面的基本结构：

`<el-container>`：包装容器。对于移动布局，它默认垂直排列子项。当需要并排布局时，设置 `direction="horizontal"`。

`<el-header>`：标头容器。

`<el-aside>`：侧面部分的容器（通常是侧面导航）。

`<el-main>`：主要部件的容器。

`<el-footer>`：页脚容器。

:::tip

这些组件使用flex进行布局，因此请确保您的浏览器支持它。此外，`<el-container>`的直接子元素必须是后四个组件中的一个或多个。并且后四个组件的父元素必须是`<el-container>`。

:::

## 容器API

### 容器属性

| 名称 | 描述             | 类型                                | 默认 |
| ---- | ---------------- | ----------------------------------- | ---- |
| 方向 | 子元素的布局方向 | ^[枚举]`'horizontal' \| 'vertical'` | 垂直 |

### 容器槽位

| 名称 | 描述           | 子标签                 |
| ---- | -------------- | ---------------------- |
| 默认 | 自定义默认内容 | 容器/页眉/旁白/主/页脚 |
