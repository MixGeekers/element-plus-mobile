# 轮播

在有限的空间内循环播放一系列图像或文本

## 基本用法

:::demo 将 `el-carousel` 与 `el-carousel-item` 结合起来，你会得到一个轮播。每张幻灯片的内容都是完全可定制的，您只需将其放置在 `el-carousel-item` 标签内即可。默认情况下，当鼠标悬停在指示器上时，轮播会切换。设置`trigger`为`click`，只有点击指示器时轮播才会切换。

carousel/basic

:::

## 运动模糊^(2.6.0)

添加运动模糊，为轮播注入活力和平滑度。

:::demo 启用运动模糊可增强轮播的动态性和平滑度。默认情况下，`motion-blur` 参数设置为 `false`，激活此功能并提供视觉上引人入胜的体验。

carousel/motion-blur

:::

## 指标

指标可以显示在轮播之外

:::demo `indicator-position` 属性确定指示器所在的位置。默认情况下，它们位于轮播内部，将 `indicator-position` 设置为 `outside` 将它们移到外部；将 `indicator-position` 设置为 `none` 会隐藏指示器。

carousel/indicator

:::

## 箭头

您可以定义何时显示箭头

:::demo `arrow` 属性确定何时显示箭头。默认情况下，当鼠标悬停在轮播上时，它们就会出现。将 `arrow` 设置为 `always` 或 `never` 永久显示/隐藏箭头。

carousel/arrows

:::

## 自动高度

当`carousel`的`height`设置为`auto`时，`carousel`高度将根据`carousel item`的高度自动设置

:::demo

carousel/auto-height

:::

## 卡牌模式

当页面足够宽但高度有限时，可以激活轮播的卡片模式

:::demo 将 `type` 设置为 `card` 激活卡模式。除了外观之外，卡片模式与普通模式最大的区别就是点击两侧的幻灯片可以直接切换卡片模式的轮播。

carousel/card

:::

## 垂直

默认情况下，`direction` 为 `horizontal`。通过将`direction`设置为`vertical`，让轮播在垂直方向显示。

:::demo

carousel/vertical

:::

## 轮播 API

### 轮播属性

| 名称              | 描述                               | 类型                                    | 默认 |
| ----------------- | ---------------------------------- | --------------------------------------- | ---- | ---- |
| 高度              | 旋转木马的高度                     | ^[字符串]                               | ''   |
| 初始索引          | 最初活动幻灯片的索引（从 0 开始）  | ^[数字]                                 | 0    |
| 触发              | 指标如何触发                       | ^[枚举]`'hover' \| 'click'`             | 悬停 |
| 自动播放          | 是否自动循环播放幻灯片             | ^[布尔值]                               | 真实 |
| 间隔              | 自动循环的间隔，以毫秒为单位       | ^[数字]                                 | 3000 | 3000 |
| 指示器位置        | 指示器的位置                       | ^[枚举]`'' \| 'none' \| 'outside'`      | ''   |
| 箭头              | 当显示箭头时                       | ^[枚举]`'always' \| 'hover' \| 'never'` | 悬停 |
| 类型              | 轮播的类型                         | ^[枚举]`'' \| 'card'`                   | ''   |
| 卡片规模 ^(2.7.8) | 当类型为卡片时，辅助卡片的缩放尺寸 | ^[数字]                                 | 0.83 | 0.83 |
| 循环              | 显示循环中的项目                   | ^[布尔值]                               | 真实 |
| 方向              | 显示方向                           | ^[枚举]`'horizontal' \| 'vertical'`     | 水平 |
| 悬停暂停          | 悬停时暂停自动播放                 | ^[布尔值]                               | 真实 |
| 运动模糊 ^(2.6.0) | 为旋转木马注入活力和流畅感         | ^[布尔值]                               | 假   |

### 轮播活动

| 名称 | 描述                                                                                       | 类型                                                |
| ---- | ------------------------------------------------------------------------------------------ | --------------------------------------------------- |
| 改变 | 当活动幻灯片切换时触发，它有两个参数，一个是新活动幻灯片的索引，另一个是旧活动幻灯片的索引 | ^[功能]`(current: number, prev: number) => boolean` |

### 轮播老虎机

| 名称 | 描述           | 子标签   |
| ---- | -------------- | -------- |
| 默认 | 自定义默认内容 | 轮播项目 |

### 轮播暴露

| 方法              | 描述                                                                      | 类型                |
| ----------------- | ------------------------------------------------------------------------- | ------------------- | ------------------------------------------ |
| 活动索引 ^(2.7.8) | 活动幻灯片索引                                                            | ^[数字]             |
| 设置活动项目      | 手动切换幻灯片，要切换到的幻灯片索引，从0开始；或对应的`el-carousel-item` | 的`name`            | ^[功能]`(index: string \| number) => void` |
| 上一页            | 切换到上一张幻灯片                                                        | ^[功能]`() => void` |
| 下一个            | 切换到下一张幻灯片                                                        | ^[功能]`() => void` |

## 轮播项目 API

### 轮播项目属性

| 名称 | 描述                             | 类型                | 默认 |
| ---- | -------------------------------- | ------------------- | ---- |
| 名称 | 项目名称，可用于 `setActiveItem` | ^[字符串]           | ''   |
| 标签 | 相应指标的文本内容               | ^[字符串] / ^[数字] | ''   |

### 轮播物品槽

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
