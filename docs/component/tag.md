# 标签

用于标记和选择。

## 基本用法

:::demo 使用 `type` 属性定义标签的类型。另外，`color`属性可用于设置Tag的背景颜色。

tag/basic

:::

## 可移除标签

:::demo `closable` 属性可用于定义可移动标记。它接受 `Boolean`。默认情况下，标签的删除有一个淡入淡出的动画。如果您不想使用它，可以将接受 `Boolean` 的 `disable-transitions` 属性设置为 `true`。 `close` 事件在标签被移除时触发。

tag/removable

:::

## 动态编辑

您可以使用 `close` 事件动态添加和删除标签。

:::demo

tag/editable

:::

## 尺寸

除了默认大小之外，Tag 组件还提供了三种额外的大小供您在不同的场景中进行选择。

:::demo 使用属性 `size` 通过 `large`、`default` 或 `small` 设置附加尺寸。

tag/sizes

:::

## 主题

标签提供三种不同的主题：`dark`、`light` 和 `plain`

:::demo 使用`effect`进行更改，默认为`light`

tag/theme

:::

## 圆角

标签也可以像按钮一样是圆形的。

:::demo

tag/rounded

:::

## 可检查标签

有时因为业务需要，我们可能需要像复选框这样的标签，但是**像复选框这样的按钮**不能满足我们的需求，这里有`check-tag`。您可以在 ^(2.5.4) 中使用 `type` 属性。

:::demo 基本检查标签用法，API 比较简单。

:::

## 标签API

### 标签属性

| 名称     | 描述              | 类型                                                               | 默认 |
| -------- | ----------------- | ------------------------------------------------------------------ | ---- |
| 类型     | 标签类型          | ^[枚举]`'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | 初级 |
| 可关闭   | 是否可以删除Tag   | ^[布尔值]                                                          | 假   |
| 禁用转换 | 是否禁用动画      | ^[布尔值]                                                          | 假   |
| 击中     | Tag是否有高亮边框 | ^[布尔值]                                                          | 假   |
| 颜色     | 标签的背景颜色    | ^[字符串]                                                          | —    |
| 尺寸     | 标签尺寸          | ^[枚举]`'large' \| 'default' \| 'small'`                           | —    |
| 效果     | 标签主题          | ^[枚举]`'dark' \| 'light' \| 'plain'`                              | 光   |
| 圆形     | 标签是否圆角      | ^[布尔值]                                                          | 假   |

### 标签事件

| 名称 | 描述           | 类型                               |
| ---- | -------------- | ---------------------------------- |
| 点击 | 单击标签时触发 | ^[功能]`(evt: MouseEvent) => void` |
| 关闭 | 删除标签时触发 | ^[功能]`(evt: MouseEvent) => void` |

### 标签槽

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |

## 检查标签 API

### 检查标签属性

| 名称                    | 描述               | 类型                                                               | 默认 |
| ----------------------- | ------------------ | ------------------------------------------------------------------ | ---- |
| 已检查 / v-model:已检查 | 已检查             | ^[布尔值]                                                          | 假   |
| 已禁用 ^(2.8.2)         | 检查标签是否被禁用 | ^[布尔值]                                                          | 假   |
| 输入 ^(2.5.4)           | 检查标签的类型     | ^[枚举]`'primary' \| 'success' \| 'info' \| 'warning' \| 'danger'` | 初级 |

### 检查标签事件

| 名称 | 描述                 | 类型                              |
| ---- | -------------------- | --------------------------------- |
| 改变 | 单击“检查标签”时触发 | ^[功能]`(value: boolean) => void` |

### 检查标签槽

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
