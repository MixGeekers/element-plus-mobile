## API

### 属性

| 名称          | 描述           | 类型                                                               | 默认 |
| ------------- | -------------- | ------------------------------------------------------------------ | ---- |
| 类型          | 文本类型       | ^[枚举]`'primary' \| 'success' \| 'warning' \| 'danger' \| 'info'` | —    |
| 尺寸          | 文字大小       | ^[枚举]`'large' \| 'default' \| 'small'`                           | 默认 |
| 截断          | 渲染省略号     | ^[布尔值]                                                          | 假   |
| 线夹 ^(2.4.0) | 最大行数       | ^[字符串] / ^[数字]                                                | —    |
| 标签          | 自定义元素标签 | ^[字符串]                                                          | 跨度 |
