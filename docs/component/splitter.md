# 分离器 ^(测试版)

水平或垂直划分区域，并自由拖动以调整每个区域的大小。

## 基本用法

最基本的用法，如果没有传递默认大小，会自动均分。

:::demo

splitter/basic

:::

## 垂直

使用垂直方向。

:::demo

splitter/vertical

:::

## 可折叠

配置`collapsible`提供快速收缩能力。您可以使用 `min` 属性来防止折叠后通过拖动进行展开。

:::demo

splitter/collapsible

:::

## 禁用拖动

当任一面板禁用 `resizable` 时，拖动将被禁用。

:::demo

splitter/disableDrag

:::

## 面板尺寸

`v-model:size`可获取面板尺寸。

:::demo

splitter/size

:::

## 懒惰 ^(2.11.0)

当启用 `lazy` 时，面板尺寸在拖动过程中不会实时更新，而只会在拖动结束后实时更新。

:::demo

splitter/lazy

:::

## 分离器 API

### 分离器属性

| 名称           | 描述             | 类型                                | 默认 |
| -------------- | ---------------- | ----------------------------------- | ---- |
| 布局           | 分路器布置方向   | ^[枚举]`'horizontal' \| 'vertical'` | 水平 |
| 懒惰 ^(2.11.0) | 是否开启懒惰模式 | ^[布尔值]                           | 假   |

### 拆分事件

| 名称           | 描述                                         | 类型                                                                      |
| -------------- | -------------------------------------------- | ------------------------------------------------------------------------- |
| 调整大小开始   | 开始调整面板大小时触发，`index` 为拖动条索引 | ^[功能]`(index: number, sizes: number[]) => void`                         |
| 调整大小       | 调整面板大小时触发，`index` 是拖动条索引     | ^[功能]`(index: number, sizes: number[]) => void`                         |
| 调整大小结束   | 面板调整大小结束时触发，`index` 为拖动条索引 | ^[功能]`(index: number, sizes: number[]) => void`                         |
| 折叠 ^(2.10.3) | 当面板折叠时触发，`index` 为拖动条索引       | ^[功能]`(index: number, type: 'start' \| 'end', sizes: number[]) => void` |

## 分割面板 API

### SplitterPanel 属性

| 名称                | 描述                                   | 类型                | 默认 |
| ------------------- | -------------------------------------- | ------------------- | ---- |
| 尺寸 / v-型号：尺寸 | 面板大小（以像素或百分比为单位）       | ^[字符串] / ^[数字] | -    |
| 分钟                | 面板的最小尺寸（以像素或百分比为单位） | ^[字符串] / ^[数字] | -    |
| 最大                | 面板的最大尺寸（以像素或百分比为单位） | ^[字符串] / ^[数字] | -    |
| 可调整大小          | 面板是否可以调整大小                   | ^[布尔值]           | 真实 |
| 可折叠              | 面板是否可以折叠                       | ^[布尔值]           | 假   |

### SplitterPanel 事件

| 名称       | 描述                 | 类型                            |
| ---------- | -------------------- | ------------------------------- |
| 更新：尺寸 | 当面板大小改变时触发 | ^[功能]`(size: number) => void` |

### SplitterPanel 插槽

| 名称       | 描述                       |
| ---------- | -------------------------- |
| 默认       | 面板默认内容               |
| 开始折叠   | 开始可折叠按钮的自定义内容 |
| 末端可折叠 | 结束可折叠按钮的自定义内容 |

### SplitterPanel 暴露

| 名称                       | 描述                    | 类型                         |
| -------------------------- | ----------------------- | ---------------------------- |
| splitterPanelRef ^(2.11.9) | SplitterPanel html 元素 | ^[对象]`Ref<HTMLDivElement>` |
