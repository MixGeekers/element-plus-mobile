## 描述 API

### 描述属性

| 名称              | 描述                   | 类型                                           | 默认    |
| ----------------- | ---------------------- | ---------------------------------------------- | ------- | --- |
| 边框              | 有或无边框             | ^[布尔值]                                      | 假      |
| 专栏              |                        | 一行中有多个 `Descriptions Item`               | ^[数字] | 3   |
| 方向              | 列表方向               | ^[枚举]`'vertical' \| 'horizontal'`            | 水平    |
| 尺寸              | 列表大小               | ^[枚举]`'' \| 'large' \| 'default' \| 'small'` | —       |
| 标题              | 标题文本，显示在左上角 | ^[字符串]                                      | ''      |
| 额外              | 额外文本，显示在右上角 | ^[字符串]                                      | ''      |
| 标签宽度 ^(2.8.8) | 每列标签宽度           | ^[字符串] / ^[数字]                            | —       |

### 描述 插槽

| 名称 | 描述                         | 子标签    |
| ---- | ---------------------------- | --------- |
| 默认 | 自定义默认内容               | 描述 项目 |
| 标题 | 自定义标题，显示在左上角     | —         |
| 额外 | 自定义额外区域，显示在右上角 | —         |

## 描述项 API

### 描述项目属性

| 名称              | 描述                                                                                                                     | 类型                                      | 默认                |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------- | ------------------- | --- |
| 标签              | 标签文字                                                                                                                 | ^[字符串]                                 | ''                  |
| 跨度              | 列的 colspan                                                                                                             | ^[数字]                                   | 1                   |
| 行距 ^(2.8.1)     | 单元格应跨越的行数                                                                                                       | ^[数字]                                   | 1                   |
| 宽度              | 列宽，不同行同一列的宽度设置最大值（如果没有`border`，则宽度包含标签和内容）                                             | ^[字符串] / ^[数字]                       | ''                  |
| 最小宽度          | 列最小宽度，`width` 的列具有固定宽度，而 `min-width` 的列具有按比例分布的宽度（如果没有 `border`，则宽度包含标签和内容） | ^[字符串] / ^[数字]                       | ''                  |
| 标签宽度 ^(2.8.8) | 列标签宽度，如果不设置，将与列的宽度相同。                                                                               | `Descriptions` 的优先级高于 `label-width` | ^[字符串] / ^[数字] | —   |
| 对齐              | 列内容对齐（如果没有`border`，对标签和内容均有效）                                                                       | ^[枚举]`'left' \| 'center' \| 'right'`    | 左                  |
| 标签对齐          | 列标签对齐方式，如果省略，将应用上述`align`属性的值（如果没有`border`，请使用`align`属性）                               | ^[枚举]`'left' \| 'center' \| 'right'`    | —                   |
| 类名              | 栏目内容自定义类名                                                                                                       | ^[字符串]                                 | ''                  |
| 标签类别名称      | 列标签自定义类名                                                                                                         | ^[字符串]                                 | ''                  |

### 描述物品槽位

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
| 标签 | 定制标签       |
