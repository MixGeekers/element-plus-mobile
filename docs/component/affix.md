# 附加

将元素固定到特定的可见区域。

## 基本用法

默认情况下，Affix 固定在页面顶部。

:::demo 您可以设置 `offset` 属性来改变偏移顶部，默认值为 0。

affix/basic

:::

## 目标容器

您可以设置 `target` 属性以始终将词缀保留在容器中。如果超出范围，它将被隐藏。

:::demo 请注意容器避免有滚动条。

affix/target

:::

## 固定位置

词缀组件提供两个固定位置：`top` 和 `bottom`。

:::demo 您可以设置 `position` 属性来更改固定位置，默认值为 `top`。

affix/fixed

:::

## API

### 属性

| 名称             | 描述                                                                | 类型                          | 默认 |
| ---------------- | ------------------------------------------------------------------- | ----------------------------- | ---- | --- |
| 偏移             | 偏移距离                                                            | ^[数字]                       | 0    |
| 位置             | 词缀位置                                                            | ^[枚举]`'top' \| 'bottom'`    | 顶部 |
| 目标             | 目标容器（CSS 选择器）                                              | ^[字符串]                     | —    |
| z 索引           | 后缀`z-index`                                                       | ^[数字]                       | 100  | 100 |
| 传送 ^(2.13.0)   | 词缀元素是否被传送，如果是`true`，则会被传送到`append-to`设置的位置 | ^[布尔值]                     | 假   |
| 追加到 ^(2.13.0) | 词缀元素附加到哪个元素                                              | ^[CSS选择器] / ^[HTMLElement] | 身体 |

### 活动

| 名称 | 描述                 | 类型                                                            |
| ---- | -------------------- | --------------------------------------------------------------- |
| 改变 | 当固定状态改变时触发 | ^[功能]`(fixed: boolean) => void`                               |
| 滚动 | 滚动时触发           | ^[功能]`(value: { scrollTop: number, fixed: boolean }) => void` |

### 老虎机

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |

### 暴露

|名称 |描述 |类型 |
| ---------- | ------------------------ | | ----------------------- |
|更新 |手动更新附加状态 | ^[功能]`() => void` |
|更新根目录 |更新 rootRect 信息 | ^[功能]`() => void` |
