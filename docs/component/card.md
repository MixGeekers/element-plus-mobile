# 卡

将信息集成到卡片容器中。

## 基本用法

卡片包括标题、内容和操作。

:::demo 卡由 `header`、`body` 和 `footer` 组成。 `header`和`footer`是可选的，其内容分发取决于命名槽。

card/basic

:::

## 简单的卡片

标题部分可以省略。

:::demo

card/simple

:::

## 带图像

通过添加一些配置来显示更丰富的内容。

:::demo `body-style` 属性定义自定义 `body` 的 CSS 样式。

card/with-images

:::

## 阴影

您可以定义何时显示卡片阴影

:::demo `shadow` 属性决定何时显示卡片阴影。它可以是 `always`、`hover` 或 `never`。

card/shadow

:::

## API

### 属性

| 名称               | 描述                           | 类型                              | 默认 |
| ------------------ | ------------------------------ | --------------------------------- | ---- |
| 标题               | 卡的标题。还接受 `slot#header` | 传递的 DOM ^[字符串]              | —    |
| 页脚 ^(2.4.3)      | 卡的页脚。还接受 `slot#footer` | 传递的 DOM ^[字符串]              | —    |
| 身体风格           | 卡体CSS样式                    | ^[对象]`CSSProperties`            | —    |
| 标头类 ^(2.9.8)    | 卡头的自定义类名               | ^[字符串]                         | —    |
| 身体等级 ^(2.3.10) | 卡体自定义类名                 | ^[字符串]                         | —    |
| 页脚类 ^(2.9.8)    | 卡片页脚的自定义类名称         | ^[字符串]                         | —    |
| 影子               | 何时显示卡片阴影               | ^[枚举]`always \| never \| hover` | 总是 |

### 老虎机

| 名称 | 描述           |
| ---- | -------------- |
| 默认 | 自定义默认内容 |
| 标题 | 卡片标题的内容 |
| 页脚 | 卡片页脚的内容 |
