# 时间选择

使用“时间选择”进行时间输入。

可用时间范围为 00:00 至 23:59

## 固定时间选择器

提供固定时间列表供用户选择。

:::demo 使用 `el-time-select` 标签，然后使用 `start`、`end` 和 `step` 分配开始时间、结束时间和时间步。

time-select/basic

:::

## 时间格式

使用 `format` 控制时间（小时和分钟）的格式。

检查 [此处](https://day.js.org/docs/en/display/format#list-of-all-available-formats) Day.js 所有可用格式的列表。

:::warning

注意大小写

:::

:::demo

time-select/time-formats

:::

## 固定时间范围

如果首先选择了开始（结束）时间，那么结束（开始）时间的选项状态将相应改变。

:::demo

time-select/time-range

:::

## API

### 属性

| 名称                  | 描述                                                                             | 类型                                                                                            | 默认       |
| --------------------- | -------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- | ---------- | ----- |
| 模型值 / v 模型       | 绑定值                                                                           | ^[字符串]                                                                                       | —          |
| 已禁用                | 是否禁用 TimeSelect                                                              | ^[布尔值]                                                                                       | 假         |
| 可编辑                | 输入是否可编辑                                                                   | ^[布尔值]                                                                                       | 真实       |
| 可清除                | 是否显示清除按钮                                                                 | ^[布尔值]                                                                                       | 真实       |
| 包括结束时间 ^(2.9.3) | 选项中是否包含`end`                                                              | ^[布尔值]                                                                                       | 假         |
| 尺寸                  | 输入的大小                                                                       | ^[枚举]`'large' \| 'default' \| 'small'`                                                        | 默认       |
| 占位符                | 非范围模式下的占位符                                                             | ^[字符串]                                                                                       | —          |
| 名称 ^(2.13.3)        | 与本机输入中的 `name` 相同                                                       | ^[字符串]                                                                                       | —          |
| 效果                  | 工具提示主题，内置主题：`dark` / `light`                                         | ^[字符串] / ^[枚举]`'dark' \| 'light'`                                                          | 光         |
| 前缀图标              | 自定义前缀图标组件                                                               | ^[字符串] / ^[组件]                                                                             | 时钟       |
| 清除图标              | 自定义透明图标组件                                                               | ^[字符串] / ^[组件]                                                                             | 圆圈关闭   |
| 开始                  | 开始时间                                                                         | ^[字符串]                                                                                       | 09:00      | 09:00 |
| 结束                  | 结束时间                                                                         | ^[字符串]                                                                                       | 18:00      | 18:00 |
| 步骤                  | 时间步                                                                           | ^[字符串]                                                                                       | 00:30      | 00:30 |
| 最短时间              | 最短时间，在此时间之前的任何时间都将被禁用                                       | ^[字符串]                                                                                       | —          |
| 最大时间              | 最大时间，超过此时间的任何时间都将被禁用                                         | ^[字符串]                                                                                       | —          |
| 格式                  | 设置时间格式                                                                     | ^[字符串] 请参阅[格式](https://day.js.org/docs/en/display/format#list-of-all-available-formats) | 高度：毫米 |
| 空值 ^(2.7.0)         | 组件的空值，[参见配置提供程序](./config-provider.md#empty-values-configurations) | ^[数组]                                                                                         | —          |
| 清除值 ^(2.7.0)       | 清除返回值，[参见配置提供程序](./config-provider.md#empty-values-configurations) | ^[字符串] / ^[数字] / ^[布尔值] / ^[函数]                                                       | —          |
| 波普级 ^(2.11.4)      | TimeSelect 下拉菜单的自定义类名                                                  | ^[字符串]                                                                                       | ''         |
| 波普风格 ^(2.11.4)    | TimeSelect 下拉菜单的自定义样式                                                  | ^[字符串] / ^[对象]                                                                             | —          |

### 活动

| 名称          | 描述                    | 类型                                    |
| ------------- | ----------------------- | --------------------------------------- |
| 改变          | 当用户确认值时触发      | ^[功能]`(value: string) => void`        |
| 模糊          | 当输入模糊时触发        | ^[功能]`(event: FocusEvent) => void`    |
| 焦点          | 当输入聚焦时触发        | ^[功能]`(event: FocusEvent) => void`    |
| 清除 ^(2.7.7) | 当在可清除的 TimeSelect | 中单击清除图标时触发^[功能]`() => void` |

| 方法 | 描述         | 类型                |
| ---- | ------------ | ------------------- |
| 焦点 | 聚焦输入组件 | ^[功能]`() => void` |
| 模糊 | 模糊输入组件 | ^[功能]`() => void` |
