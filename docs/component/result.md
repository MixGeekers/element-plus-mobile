## API

### 属性

| 名称   | 描述           | 类型                                                                       | 默认 |
| ------ | -------------- | -------------------------------------------------------------------------- | ---- |
| 标题   | 结果标题       | ^[字符串]                                                                  | ''   |
| 副标题 | 结果子标题     | ^[字符串]                                                                  | ''   |
| 图标   | 结果的图标类型 | ^[枚举]`'primary' (2.9.11) \| 'success' \| 'warning' \| 'info' \| 'error'` | 信息 |

### 老虎机

| 名称   | 描述                 |
| ------ | -------------------- |
| 图标   | 内容作为结果图标     |
| 标题   | 内容作为结果标题     |
| 副标题 | 内容作为结果副标题   |
| 额外   | 内容作为结果额外区域 |
